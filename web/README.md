# Run Project

#### --Front-end :computer:
```bash
cd web
```
```bash
npm install
```
```bash
npm start
```
#### --Server :globe_with_meridians:
```bash
cd web `(yes, the web directory)`
```
```bash
npm run server OR yarn server
```
